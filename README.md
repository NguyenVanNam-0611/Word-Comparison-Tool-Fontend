# Word-Comparison-Tool-Fontend
Đây là phần frontend của dự án Word Document Comparison Backend, được mình xây dựng để hiển thị kết quả so sánh tài liệu Word một cách trực quan. Frontend hỗ trợ upload hai file Word, theo dõi job xử lý từ backend, nhận dữ liệu JSON kết quả diff và render giao diện so sánh song song giữa bản gốc và bản đã chỉnh sửa.
